# Card Vault Customs — Website Launch Kit

Everything you need to launch your Shopify store for custom Pokémon & One Piece graded-style card slabs. Work through the files in order. Each one is written to **copy-paste straight into Shopify**.

> **Currency:** CAD · **Shipping from:** Canada · **Platform:** Shopify Basic

## Start here

1. **`01-SETUP-RUNBOOK.md`** — Your step-by-step build guide. Follow this top to bottom.
2. **`02-PRODUCTS.md`** — Product titles, descriptions, prices, and the photo-upload setup.
3. **`03-PAGES.md`** — Homepage sections, About, How It Works, and FAQ copy.
4. **`04-POLICIES.md`** — Shipping, returns, privacy, terms + the important "custom novelty" disclaimer.
5. **`05-BRAND-CHECKLIST.md`** — Logo, colors, fonts, and connecting Instagram.

## The 30-second overview of the build

| Step | What | Where in Shopify |
|------|------|------------------|
| 1 | Buy domain + start trial | Shopify signup |
| 2 | Pick free "Dawn" theme | Online Store → Themes |
| 3 | Add products + prices | Products |
| 4 | Install photo-upload app | Apps |
| 5 | Turn on Shopify Payments | Settings → Payments |
| 6 | Set shipping zones (CA + intl) | Settings → Shipping |
| 7 | Paste in pages & policies | Online Store → Pages / Policies |
| 8 | Connect Instagram | Sales channels |
| 9 | Test order + go live | — |

## ⚠️ Read before you sell anything

These are **custom fan-art novelty items in display slabs** — they are NOT authentic PSA/BGS/CGC graded cards, and they are NOT official Pokémon or One Piece products. The disclaimer in `04-POLICIES.md` is there to protect you legally. Keep it on the site. See that file for details.
