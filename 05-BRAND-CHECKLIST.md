# 05 · Brand & Launch Checklist

## Branding basics

- **Logo:** If you have one on Instagram, export it at high res (transparent PNG). No logo yet? A clean wordmark "CARD VAULT CUSTOMS" in a bold font works great — free tools: Canva, or Shopify's built-in logo maker (Hatchful).
- **Colors:** Pick 2–3 and use them everywhere (theme, IG, packaging). A "vault/premium" feel suggests deep navy or black + a metallic gold/silver accent. Set these in Theme editor → Settings → Colors.
- **Fonts:** Dawn's defaults are clean. If customizing, pair one bold display font (headings) with a simple sans-serif (body).
- **Product photos:** Shoot slabs on a clean, consistent background with good lighting. Consistency makes the store look pro. Same angle/background for every listing.
- **Packaging:** Cheap branded touch = a thank-you card + sticker in every mailer. Drives repeat orders and IG tags.

## Pre-launch checklist

- [ ] Currency set to CAD (before adding products)
- [ ] Domain connected (`cardvaultcustoms.com`)
- [ ] All 5 products added with photos, prices, and upload fields
- [ ] Collections created (Pokémon, One Piece, Quick Buy, Bundles)
- [ ] Photo-upload app installed and tested
- [ ] Shopify Payments active + test transaction works
- [ ] Shipping zones set (Canada free-over-$50, US, International)
- [ ] About / How It Works / FAQ pages live
- [ ] Refund, Shipping, Privacy, Terms policies pasted in
- [ ] Disclaimer on product pages + footer + FAQ
- [ ] Instagram sales channel connected; link in IG bio
- [ ] Test order placed (photo upload received, email sent, label prints)
- [ ] Store password removed → LIVE

## First-week growth moves

- Announce launch on Instagram with a launch code (e.g. `LAUNCH15`).
- Post the "How It Works" 3-step as a carousel/reel.
- Run the proven **Buy 2 Get 1 Free** promo (Shopify → Discounts).
- Ask your first few customers for a photo + quote to seed reviews.
- Add a product review app (free: **Shopify Product Reviews** or **Judge.me**) so social proof builds automatically.
